# 일별 입력 키보드 단축키 추가

엑셀처럼 키보드만으로 빠르게 데이터를 입력할 수 있도록 개선합니다.

## 작업 분량

- **파일 1개만 변경**
- 작업 시간: 약 10분

## 새 기능

### ⌨️ 키보드 단축키

| 키 | 동작 |
|----|------|
| **Enter** | 같은 날짜의 아래 항목으로 이동 |
| **↓ (아래 화살표)** | 같은 날짜의 아래 항목으로 이동 |
| **↑ (위 화살표)** | 같은 날짜의 위 항목으로 이동 |
| **Tab** | 다음 날짜로 이동 |
| **Shift + Tab** | 이전 날짜로 이동 |
| **→ (오른쪽 화살표)** | 다음 날짜 (커서가 끝에 있을 때만) |
| **← (왼쪽 화살표)** | 이전 날짜 (커서가 시작에 있을 때만) |

### 보너스 기능

- **자동 텍스트 선택**: 칸으로 이동하면 기존 숫자가 자동 선택됨 → 바로 새 숫자 입력 가능
- **마지막 칸 도달 시**: 자동으로 첫 번째 칸으로 순환 이동
- **자동 저장**: 키보드 이동 중에도 모든 변경 자동 저장

## 작업 단계

### Step 1: 파일 다운로드 및 교체

1. `page.tsx` 다운로드
2. 윈도우 탐색기에서 GitHub seyang 폴더 이동:
   ```
   C:\Users\USER\Documents\GitHub\seyang\src\app\team\[code]\
   ```
3. 그 폴더 안의 **`page.tsx`** 를 새 파일로 **덮어쓰기**

   주의: `manage` 폴더 안의 `page.tsx`나 `dashboard` 폴더 안의 `page.tsx`가 아닙니다. 
   바로 `[code]` 폴더 안의 `page.tsx`입니다.

### Step 2: GitHub Desktop으로 푸시

1. GitHub Desktop 열기
2. 변경 파일 1개 확인: `src/app/[code]/page.tsx`
3. Summary: `Add keyboard shortcuts for daily input`
4. **Commit to main**
5. **Push origin**

### Step 3: Vercel 자동 배포 대기 (1~2분)

### Step 4: 테스트

1. 사이트 접속 → 염지팀 (또는 다른 팀)
2. 첫 번째 날짜의 첫 번째 칸 클릭 (예: 4/27 일 총 생산시간)
3. 숫자 입력
4. **Enter 키 누르기** → 자동으로 같은 날짜의 "조미 횟수" 칸으로 이동!
5. 또 입력 후 Enter → "메추리알 수율" 이동
6. 마지막 항목까지 가면 → 다음 날짜 (4/28) 첫 항목으로 이동

화면 하단에 키보드 단축키 안내가 작게 표시됩니다.

## 입력 흐름 예시

기존 방식:
```
4/27 일 총 생산시간 클릭 → 220 입력 → 4/27 조미 횟수 클릭 → 4 입력 → 4/27 메추리알 수율 클릭 → 24 입력 → ...
(매번 마우스로 클릭)
```

새 방식:
```
4/27 일 총 생산시간 클릭 (한 번만!) → 220 → Enter → 4 → Enter → 24 → Enter → ... 
(이후 키보드만으로 모든 입력)
```

→ 입력 시간 **70% 단축** 가능!

## 추가로 개선 가능한 것

이번 변경으로 입력 속도가 빨라지면, 다음과 같은 것도 추가할 수 있습니다:

1. **숫자 입력 시 자동 검증**: 비정상 값 (예: 음수, 너무 큰 값) 즉시 경고
2. **이전 주 데이터 복사**: "지난 주와 동일" 버튼
3. **일괄 입력**: 여러 날짜에 같은 값 한꺼번에 입력
4. **음성 입력**: 모바일에서 마이크로 숫자 입력

필요하시면 말씀해주세요!
